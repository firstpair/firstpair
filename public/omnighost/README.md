# Omnighost for First Pair Press

*Omnighost for First Pair Press*, by Alexy Khrabrov, is the First Pair Press
field guide to publishing from Obsidian to Ghost across every blog. It covers
mobile and desktop Obsidian workflows, image publishing, textpack import, and
multi-blog distribution.
