# History of the Rio-Grandense Republic Preview

This folder is the public preview package for *History of the Rio-Grandense
Republic: A Revised English Reading Edition*.

Dedicated preview URL:

```text
/rio-grande/preview/
```

The package is intentionally limited. It contains the original facsimile title
page with a First Pair Press translator overprint, title/front matter, the full
table of contents, and the prefatory opening text. It does not contain the
complete book.

## Included Files

- PDF preview:
  `/rio-grande/pdf/`
- EPUB preview:
  `/rio-grande/epub/`
- Online reader:
  `/read/rio-grande/`
- Chapter reader:
  `/read/rio-grande/chapters/`
- Cover asset:
  `assets/rio-grande-cover.png`
- Lagoon headboard:
  `../rio-grande-headboard.png`
- Detailed preview manifest:
  `PREVIEW.md`

Rio Grande currently has one public reading edition. South, East, West,
Rosetta, and Rosetta Scored variants are not included for this title because
those source editions do not exist for the Rio Grande book.

See [PREVIEW.md](PREVIEW.md) for word counts, checksums, source commits, and
validation results.
